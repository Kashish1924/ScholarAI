# ScholarAI

ScholarAI is an AI-ready scholarship finder portal for Indian B.Tech students. This repository currently contains the production-style Flask scaffold for Module 1, including app factory architecture, blueprint separation, CSRF protection, database extension wiring, and AI placeholder services.

## Current Status

Module 1 is complete:

- Flask app factory
- Blueprint-based modular routing
- Bootstrap-ready template structure
- Static asset structure
- Placeholder AI service functions
- Configuration and extension setup

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open `http://127.0.0.1:5000/` in your browser.
