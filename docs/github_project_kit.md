# ScholarAI GitHub Project Kit

## GitHub Repository Description

ScholarAI is a Flask-based scholarship finder portal for Indian B.Tech students with backend eligibility scoring, admin management, REST APIs, and AI-ready placeholders.

## Short Tagline Options

- AI-ready scholarship discovery with backend-first eligibility logic
- Flask scholarship portal with recommendations, admin tools, and REST APIs
- Smart scholarship finder for Indian B.Tech students

## Branch Strategy

- `main`: stable, demo-ready branch
- `develop`: integration branch for ongoing work
- `feature/<module-name>`: one branch per module or focused enhancement
- `fix/<issue-name>`: bug-fix branch
